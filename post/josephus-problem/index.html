约瑟夫环的数学解法

2012-12-22
编程笔记

文章目录

int Josephus(int n, int m)//总人数为n，从第一人开始数到m的退出，
{
    int i, r = 0;
    for (i = 2; i <= n; i++)
        r = (r + m) % i;
    return r+1;//最后一个人的初始位置
}
